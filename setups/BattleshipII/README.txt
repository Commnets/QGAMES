The default keys to manage the game are:
A: Left
S: Right
P: Forward
L: Backward
M: Fire
Z: Pause / Continue
The keys above can be configured.
The joystick is also allowed instead of the keyboard.
When playing ESC can be pressed to activate a menu for: save, continue playing or simply exit the game. 
The main character will not die when this menu is on top, but the game will continue.
