The keys to manage the game are:
A: Going left
S: Going right
P: Going up
L: Going down
W: Get / Leave things
Q: Iterate over the things being carried in the pocket
M: Use the last thing in the pocket. If it is a weapon a shooting will be fired. It also used to swim when the ninja is in the water
SPACE: Jump
Z: Pause / Restart the game
X: Save the status of the game
The keys above can be configured.
The joystick is also allowed instead of the keyboard.
When playing ESC can be pressed to activate a menu for: save, continue playing or simply exit the game. 
The main character will not die when this menu is on top, but the game will continue.
